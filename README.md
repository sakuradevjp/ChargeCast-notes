## What is ChargeCast?

A Windows app that mirrors your Android phone screen to PC over USB **while keeping the phone charged the whole time**. Built for mobile game streamers who don't want to spend on a capture card.

- 🎮 USB capture, low latency, no Wi-Fi
- 🔋 Charges your phone while you stream — no battery drain
- 🎚️ 3-channel audio mixer (device, mic, output)
- 🎨 720p / 1080p / 1440p presets for Twitch and YouTube
- 🌐 22 languages

**Get it on Microsoft Store**: https://apps.microsoft.com/detail/9n87x47v634v

**Read the docs (live page)**: https://sakuradevjp.github.io/ChargeCast-notes/
