USB 케이블 한 가닥.
Android 화면도, 게임 사운드도, 내 마이크도 통째로 Windows로 — 그것도 충전되는 내내.

▶ Microsoft Store에서 받기 7일 무료 체험 포함

이런 경험, 있으시죠?

  • Android 게임을 OBS로 방송하고 싶은데, 화면 캡처, 게임 사운드, 내 마이크가 매번 깔끔히 맞물리지 않는다.
  • Wi-Fi 미러링은 고행길: 지연은 튀고, 배터리는 쭉쭉 빠진다.
  • 명령줄에서 scrcpy를 써도 그 루틴이 매 방송마다 슬슬 지겨워진다.
  • 방송 중에 폰은 12%, 게다가 뜨끈뜨끈. 식은땀.

ChargeCast는 이 네 가지를 USB 한 가닥과 친절한 GUI로 한방에 해결합니다. Windows 전용, 방송 우선 설계.

3분 만에 방송 시작 — 빠른 데모

"나도 방송할 수 있을까"에서 "지금 방송 중"까지, 가장 짧은 길.

1USB 케이블을 꽂는다

Android 폰에서 USB 디버깅을 켜고, PC에 케이블로 연결하세요.
평소 쓰는 충전 케이블이면 됩니다. 네 — 방송 내내 폰은 충전됩니다.

처음이라면: USB 디버깅 켜기 (2분)
  1. Android 설정 → 휴대전화 정보 열기
  2. 빌드 번호를 7번 연속 탭 ("개발자가 되었습니다"가 뜰 때까지)
  3. 설정 → 시스템 → 개발자 옵션으로 이동해 USB 디버깅 켜기
  4. 케이블을 꽂았을 때 나오는 권한 팝업에서 허용을 탭

메뉴 경로는 제조사마다 조금씩 다릅니다 (삼성: 설정 → 휴대전화 정보 → 소프트웨어 정보 → 빌드 번호 탭).

2프리셋을 고르고 Start Capture

720p / 1080p / 1440p / Custom을 원클릭으로 선택. scrcpy 창이 열리고 그 안에 Android 화면이 바로 들어옵니다. OBS의 창 캡처로 집으면 — 끝.

3세 개의 슬라이더로 믹스

  • Device — Android 폰에서 나오는 게임 사운드
  • Output — PC 오디오 (방송 BGM, Discord, 브라우저)
  • Mic — 내 목소리

ChargeCast 안에서 모두 섞어 OBS에는 깔끔한 소스 하나만 넘겨줍니다. 방송 중에 믹서와 씨름할 일 없음.

여기까지 콜드 스타트에서 대략 3분. 이후엔 OBS의 "방송 시작"을 누르기만 하면 됩니다.

이런 분께 맞아요

✅ 찰떡

  • OBS로 Android 게임 방송, 앱 리뷰, 튜토리얼을 내보내고 싶은 분
  • Wi-Fi의 들쭉날쭉함보다 유선의 안정성을 선호하는 분
  • 오디오를 앱 하나로 끝내고 싶은 크리에이터
  • 장시간 방송에서 발열과 배터리가 걱정되는 분

⏭ 어쩌면 별로일지도

  • Mac 또는 Linux 사용자 (이 앱은 Windows 전용)
  • 100% 무료를 원하는 분 (명령줄 scrcpy로 충분합니다)
  • 무선 Wi-Fi 미러링이 꼭 필요한 분 (ChargeCast는 USB 우선)

자주 묻는 질문

루팅이나 특수한 개발 환경이 필요한가요?

아니요. Android 개발자 옵션에서 "USB 디버깅"만 켜면 됩니다.

게임 사운드와 내 목소리를 동시에 들을 수 있나요?

네. 세 채널마다 개별 슬라이더가 있습니다.

어떤 스트리밍 소프트웨어를 써야 하나요?

기본 설정은 OBS Studio입니다 (창 캡처 소스 사용). XSplit 등 유사한 툴도 같은 방식으로 동작합니다.

세로 화면 게임은요?

세로/가로/자동으로 방향을 고정할 수 있어 OBS 레이아웃을 안정적으로 유지할 수 있습니다.

몰래 서버에 연결하나요?

모든 캡처는 로컬 ADB를 통해 이루어집니다. ChargeCast 자체는 기기 정보를 외부로 보내지 않습니다.

OBS 연동에 대한 메모

ChargeCast는 그 자체로 방송 앱이 아닙니다. OBS에 깔끔한 재료를 넘겨주는 무대쯤으로 생각해 주세요.

  • ChargeCast 창에 OBS Studio의 창 캡처를 대기만 하면 — 연동은 그게 전부
  • 영상, 기기 사운드, 마이크는 이미 ChargeCast 쪽에서 믹스되어 있어 OBS 씬은 최소한으로 유지됩니다
  • XSplit, Streamlabs, Discord 화면 공유 — 창 캡처가 되는 도구라면 모두 같은 방식

OBS 씬을 한 번 꾸려 두면, 이후엔 단 한 동작으로 방송 시작: 케이블 꽂고 Start Capture.
