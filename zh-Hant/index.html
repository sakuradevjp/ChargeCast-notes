這些狀況是不是很熟？

  • 想在 OBS 上直播 Android 遊戲，又不想為此再買一張擷取卡或 Elgato，但畫面擷取、遊戲音和自己的麥克風每次都兜不攏。

ChargeCast 用 一條 USB 線加一個順手的 GUI 一次解決這四件事。僅限 Windows，專為直播打造。

3 分鐘開播 —— 快速示範

從「說不定我也能直播」到「我已經在直播了」的最短路徑。

1插上 USB 線

在 Android 手機上開啟 USB 偵錯，然後用線接上 PC。
平常的充電線就可以。 沒錯 —— 直播全程手機都在充電。

第一次用？啟用 USB 偵錯（2 分鐘）
  1. 開啟 Android 設定 → 關於手機
  2. 連續點擊 版本號碼 7 次，直到出現「您已成為開發人員」
  3. 前往 設定 → 系統 → 開發人員選項，打開 USB 偵錯
  4. 插上線後，在權限提示上點 允許

各家手機選單路徑略有差異（三星：設定 → 關於手機 → 軟體資訊 → 點擊版本號碼）。

2選個預設，按 Start Capture

一鍵選擇 720p / 1080p / 1440p / Custom。scrcpy 視窗會彈出，Android 畫面已經在裡面了。把 OBS 的視窗擷取對準它 —— 搞定。

3三個滑桿混音搞定

  • Device — 來自 Android 手機的遊戲聲
  • Output — 你的 PC 音訊（直播 BGM、Discord、瀏覽器）
  • Mic — 你自己的聲音

所有聲音都在 ChargeCast 裡混好，再給 OBS 一路乾淨的來源。不必在直播途中跟混音器打架了。

從冷開機到這裡大約 3 分鐘。接下來只要按 OBS 的「開始直播」即可。

常見問題

需要 root 或特殊的開發環境嗎？

不用。只要在 Android 開發人員選項裡打開「USB 偵錯」即可。

能同時聽到遊戲聲和自己的聲音嗎？

可以。三個聲道各自有獨立滑桿。

該搭配哪款直播軟體？

OBS Studio 是預設搭配（請用「視窗擷取」來源）。XSplit 等軟體用法相同。

直式畫面的遊戲呢？

方向可以鎖定為直式、橫式或自動，這樣 OBS 配置就能保持穩定。

它會偷偷連網嗎？

所有擷取都走本機 ADB。ChargeCast 本身不會把裝置資訊送到任何地方。

和 scrcpy 或 DroidCam 有什麼差別？

ChargeCast 把 scrcpy 加上 3 聲道音訊混音器和一鍵 GUI 打包在一起，讓遊戲音和麥克風以一路乾淨的來源進入 OBS。原生 scrcpy 不一定能傳遊戲音（僅 Android 11+ 且有限制），而 DroidCam 把手機當成網路攝影機，並不是針對螢幕鏡像而設計。ChargeCast 是圍繞直播主的實際流程做的：插 USB、邊充電、OBS 視窗擷取。

關於與 OBS 整合的說明

ChargeCast 本身不是直播軟體。可以把它想成一個舞台，把乾淨的畫面交給 OBS。

  • 讓 OBS Studio 的視窗擷取 對準 ChargeCast 視窗 —— 整合就這一步
  • 視訊、裝置音與麥克風已在 ChargeCast 端混好，所以 OBS 場景可以保持精簡
  • XSplit、Streamlabs、Discord 畫面分享 —— 只要能用視窗擷取的環境，用法都一樣

OBS 場景只要設定好一次，之後開播只要一個動作：插上線，按 Start Capture。
