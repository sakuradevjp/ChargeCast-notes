Kommt dir das bekannt vor?

  • Du willst Android-Gameplay in OBS streamen ohne dafür eine Capture Card oder einen Elgato kaufen zu müssen, aber Bildschirmaufnahme, Spielsound und dein eigenes Mikro passen nie sauber zusammen.
  • Wi-Fi-Mirroring ist ein Albtraum: Latenz-Spitzen und ein leerer Akku.

ChargeCast löst alle vier Probleme mit einem USB-Kabel und einer freundlichen GUI. Nur für Windows, auf Streaming ausgelegt.

In 3 Minuten live — Quick Demo

Der kürzeste Weg von „Vielleicht könnte ich streamen“ zu „Ich bin live.“

1USB-Kabel einstecken

Schalte USB-Debugging auf deinem Android-Handy ein und steck es in den PC.
Dein ganz normales Ladekabel reicht völlig. Ja — das Handy lädt, während du streamst.

Zum ersten Mal? USB-Debugging aktivieren (2 Minuten)
  1. Öffne Android Einstellungen → Über das Telefon
  2. Tippe 7 Mal auf Buildnummer, bis „Sie sind jetzt Entwickler“ erscheint
  3. Geh zu Einstellungen → System → Entwickleroptionen und aktiviere USB-Debugging
  4. Wenn du das Kabel einsteckst, tippe bei der Berechtigungsanfrage auf Zulassen

Die Menüpfade variieren je nach Hersteller leicht (Samsung: Einstellungen → Über das Telefon → Softwareinformationen → Buildnummer antippen).

2Preset wählen, Start Capture drücken

Wähle 720p / 1080p / 1440p / Custom mit einem Klick. Ein scrcpy-Fenster öffnet sich mit deinem Android-Bildschirm drin. Richte die Fensteraufnahme von OBS darauf — fertig.

3Mit drei Reglern mischen

  • Device — Spielsound von deinem Android-Handy
  • Output — dein PC-Audio (Stream-BGM, Discord, Browser)
  • Mic — deine eigene Stimme

Mische alles in ChargeCast und reiche OBS eine einzige saubere Quelle rüber. Kein Mixer-Chaos mehr mitten im Stream.

Das sind grob 3 Minuten vom Kaltstart. Danach musst du nur noch in OBS auf „Stream starten“ drücken.

FAQ

Brauche ich Root oder eine besondere Entwicklerumgebung?

Nein. Einfach „USB-Debugging“ in den Android-Entwickleroptionen einschalten.

Kann ich Spielsound und meine eigene Stimme gleichzeitig hören?

Ja. Jeder der drei Kanäle hat seinen eigenen Regler.

Welche Streaming-Software sollte ich nehmen?

OBS Studio ist das Referenz-Setup (nimm eine Fensteraufnahme-Quelle). XSplit und ähnliche Tools funktionieren genauso.

Und bei Hochformat-Spielen?

Die Ausrichtung lässt sich auf Portrait, Landscape oder Auto festlegen, damit dein OBS-Layout stabil bleibt.

Funkt es nach Hause?

Die komplette Erfassung läuft lokal über ADB. ChargeCast selbst sendet keine Geräteinfos irgendwohin.

Wie unterscheidet sich das von scrcpy oder DroidCam?

ChargeCast bündelt scrcpy mit einem 3-Kanal-Audiomixer und einer Ein-Klick-GUI, sodass Spielsound und dein Mikro als eine saubere Quelle in OBS landen. Reines scrcpy kann den Spielsound nicht immer mitnehmen (nur Android 11+ und mit Einschränkungen), und DroidCam behandelt das Handy wie eine Webcam — nicht als Bildschirmspiegel. ChargeCast ist um den Streamer-Workflow herum gebaut: USB einstecken, laden, OBS-Fensteraufnahme.

Ein Wort zur OBS-Integration

ChargeCast ist für sich allein keine Streaming-App. Stell dir das Ganze als die Bühne vor, die OBS eine saubere Aufnahme überreicht.

  • Richte eine OBS Studio Fensteraufnahme auf das ChargeCast-Fenster — das ist die ganze Integration
  • Video, Gerätesound und Mikro sind auf ChargeCast-Seite bereits gemischt, deine OBS-Szene bleibt also minimal
  • XSplit, Streamlabs, Discord-Bildschirmfreigabe — alles mit einer Fensteraufnahme-Quelle funktioniert nach demselben Prinzip

Ist die OBS-Szene einmal eingerichtet, startet jede Session mit einer einzigen Bewegung: Kabel einstecken, Start Capture drücken.

Direkte .exe-Installation

Der direkte Download ist dasselbe ChargeCast wie aus dem Microsoft Store, jedoch als eigenständiger Inno-Setup-Installer statt MSIX paketiert. Es installiert sich nach %LOCALAPPDATA%\Programs\ChargeCast pro Benutzer (ohne Administrator-Eingabeaufforderung).

Der Installer ist unsigniert, daher warnt Windows SmartScreen beim ersten Ausführen:

  1. Im blauen Dialog „Der Computer wurde durch Windows geschützt“ auf „Weitere Informationen“ klicken
  2. Auf „Trotzdem ausführen“ klicken

Um zu prüfen, ob der Installer mit der offiziellen Version übereinstimmt, vergleiche den SHA256-Hash auf der Releases-Seite: 

Get-FileHash -Algorithm SHA256 .\ChargeCast-1.7.1-Setup.exe

Pro-Funktionen (1080p / 1440p, 60fps, 3-Kanal-Audio-Mixer mit Mikrofon) werden durch eine aktive Microsoft-Store-Lizenz freigeschaltet. Ein direkter Lizenzschlüssel-Kaufkanal ist in Arbeit.
